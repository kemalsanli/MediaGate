MediaGate는 지원되지 않는 미디어 파일을 변환하여 사진 라이브러리에 직접 저장합니다. 어떤 앱에서든 공유하세요 — 파일, Safari, 메일, 메시지 — 나머지는 MediaGate가 처리합니다.

클라우드 업로드 없음. 계정 없음. 추적 없음. 모든 것이 기기에서 처리됩니다.

▶ 사용 방법
파일의 공유 버튼을 탭하고 ‘MediaGate로 저장’을 선택하면 변환된 파일이 몇 초 내에 사진에 나타납니다.

▶ 지원 포맷

동영상 → MP4
MKV, AVI, WMV, WebM, FLV, MPEG, 3GP, TS, VOB, OGV, RM, DIVX 등.

이미지 → PNG / JPEG
WebP, BMP, TIFF, SVG, PSD, TGA, RAW (CR2, NEF, ARW, DNG, ORF, RW2, RAF, PEF 등).

패스스루
MP4, MOV, HEVC, JPG, PNG, HEIF, HEIC, GIF, AVIF.

3개 카테고리에 58개 파일 확장자.

▶ 주요 기능

• VideoToolbox를 통한 하드웨어 가속 인코딩
• 공유 시트에서 작동 — 앱을 열 필요 없음
• 일괄 변환 — 한 번에 최대 20개 파일
• 조절 가능한 이미지 품질 및 동영상 품질 프리셋
• 메타데이터 보존 또는 위치 데이터 제거
• 호환 파일 압축
• 앱 내 언어 전환으로 36개 언어
• 네트워크 요청 제로 — 완전 오프라인
• 광고 없음, 구독 없음

▶ 개인정보
MediaGate는 추가 전용 사진 라이브러리 접근을 사용합니다. 기존 사진을 읽을 수 없습니다. 분석 없음, 텔레메트리 없음, 타사 SDK 없음.

▶ 오픈소스
MediaGate는 GPLv3 하에 무료 오픈소스입니다. 소스 코드는 GitHub에서 확인할 수 있습니다.

개발을 지원하고 싶다면 선택적 팁 자를 이용할 수 있습니다.
