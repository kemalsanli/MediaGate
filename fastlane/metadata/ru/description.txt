MediaGate конвертирует неподдерживаемые медиафайлы и сохраняет их напрямую в вашу медиатеку Фото. Поделитесь из любого приложения — Файлы, Safari, Почта, Сообщения — MediaGate сделает остальное.

Без загрузки в облако. Без аккаунта. Без слежки. Всё происходит на вашем устройстве.

▶ КАК ЭТО РАБОТАЕТ
Нажмите кнопку «Поделиться» на любом файле, выберите «Сохранить через MediaGate», и конвертированный файл появится в Фото за секунды.

▶ ПОДДЕРЖИВАЕМЫЕ ФОРМАТЫ

Видео → MP4
MKV, AVI, WMV, WebM, FLV, MPEG, 3GP, TS, VOB, OGV, RM, DIVX и др.

Изображение → PNG / JPEG
WebP, BMP, TIFF, SVG, PSD, TGA, RAW (CR2, NEF, ARW, DNG, ORF, RW2, RAF, PEF и др.).

Прямой пропуск
MP4, MOV, HEVC, JPG, PNG, HEIF, HEIC, GIF, AVIF.

58 расширений файлов в 3 категориях.

▶ ВОЗМОЖНОСТИ

• Аппаратное ускорение через VideoToolbox
• Работает из меню «Поделиться» — не нужно открывать приложение
• Пакетная конвертация — до 20 файлов одновременно
• Настраиваемое качество изображений и пресеты видео
• Сохранение или удаление метаданных и геолокации
• Сжатие совместимых файлов
• 36 языков с переключением в приложении
• Ноль сетевых запросов — полностью офлайн
• Без рекламы, без подписки

▶ КОНФИДЕНЦИАЛЬНОСТЬ
MediaGate использует доступ только для добавления. Не может читать ваши существующие фото. Без аналитики, без телеметрии, без сторонних SDK.
