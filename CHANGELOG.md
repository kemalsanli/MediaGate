# Changelog

All notable changes to MediaGate will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/).

## [1.0.0] - 2026-04-12

### Added
- Universal media converter with Share Extension and Action Extension
- Video conversion (MKV, AVI, WMV, FLV, WebM, 3GP, MPEG, TS, VOB, OGV, RM) to MP4
- Image conversion (BMP, WebP, TIFF, SVG, TGA, ICO, PSD, RAW, PCX, PPM) to PNG/JPEG
- Passthrough for natively supported formats (MP4, MOV, JPEG, PNG, HEIF, GIF, AVIF)
- In-extension conversion for small files, main app fallback for large files
- Hardware-accelerated H.264 encoding via VideoToolbox
- Tip jar with StoreKit 2 (small, medium, big)
- Localization: 36 languages
- In-app language switching
- Configurable video/image quality settings
- Metadata preservation with optional location stripping
- Crash-safe architecture with shared container queue
